02 Role of Sensors

MPU6050

IMU — INERTIAL MEASUREMENT UNIT

The primary motion sensor that measures both linear acceleration and rotational movement across 3 axes, giving us full orientation data.

Section 03
03 Role of ESP32 — The Brain
ESP32

The ESP32 microcontroller acts as the onboard computer — collecting sensor data, processing orientation calculations, and broadcasting telemetry over WiFi.

  • Collects data from all sensors via I²C / analog pins
  • Processes raw readings into usable metrics
  • Calculates Roll, Pitch, and Yaw angles
  • Transmits data to laptop via WiFi (TCP/UDP)
  • Controls the 3D visualization update loop
📌 Simple Line:  "ESP32 works like the onboard computer of a satellite."
Section 04
04 Actuator Concept

No physical actuator is used in this project. Instead, the 3D visualization serves as a virtual actuator — showing system response digitally.

⚙️ REAL SATELLITES — Physical Actuators

  • Reaction Wheels → rotate satellite attitude
  • Thrusters → change orbital position
  • Magnetorquers → use Earth's magnetic field
  • Solar panel drives → track sun orientation

💻 THIS PROJECT — Virtual Actuator

  • 3D cube rotates on screen in real-time
  • Orientation data drives visual feedback
  • System response is fully digitalized
  • No moving parts — zero hardware cost
📌 Smart Explanation:  "The system response is visualized digitally instead of using physical actuators — achieving the same conceptual outcome."
